# Better Obsidian Kanban Plugin

Create markdown-backed Kanban boards in [Obsidian](https://obsidian.md/) with enhanced features.


## Features

- Markdown-backed Kanban boards stored as regular `.md` files
- Drag-and-drop interface for managing tasks
- Support for tags, dates, and metadata
- Customizable columns with colors

## Installation

You can install this plugin manually from the GitHub releases page.

1. Go to the [Releases](https://github.com/rogr3/better-obsidian-kanban/releases) page of this repository.
2. Download the latest release zip file or the three files:
   - `main.js`
   - `manifest.json`
   - `styles.css`
3. In your Obsidian vault, open the `.obsidian/plugins/` folder (create it if it doesn’t exist).
4. Create a new folder named `better-obsidian-kanban` (or any name you prefer).
5. Copy the three files into that folder.
6. Restart Obsidian.
7. Go to **Settings → Community Plugins → Installed Plugins** and enable **Better Obsidian Kanban**.

You should now see the enhanced Kanban plugin available in Obsidian.
